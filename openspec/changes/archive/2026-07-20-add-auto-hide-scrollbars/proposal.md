# Change: 增加自动隐藏滚动条

## Why

目录树和日志正文的原生滚动条会持续占用视觉空间，尤其是同时出现横向、纵向滚动条时干扰日志阅读。滚动条应只在用户实际滚动时提供位置反馈，停止操作后自然退出视线。

## What Changes

- 默认隐藏目录树、日志正文、选项卡折叠菜单和可滚动弹窗的滚动条滑块与轨道。
- 任一可滚动区域发生滚动时立即显示对应滚动条；持续滚动期间保持显示。
- 停止滚动一段短暂时间后让滚动条渐隐，而不是突然消失。
- 横向与纵向滚动条采用相同行为，不改变滚动范围、虚拟列表、鼠标滚轮、触控板、键盘或拖动操作。
- 在“减少动态效果”系统偏好开启时取消渐隐动画，但仍保留自动隐藏行为。

## Impact

- Affected specs: `application-interface`（新增）
- Affected code: `src/App.tsx`、`src/styles.css`、新增滚动状态工具及测试
- Dependencies: 不新增 npm 或 Cargo 依赖
