## ADDED Requirements

### Requirement: 中英文界面本地化

系统 SHALL 为应用自有的用户界面文案提供简体中文和英文翻译，包括主窗口、目录与右键菜单、通知、设置与更新、确认提示、日志查看状态、文件选择器、可访问性标签及原生托盘菜单。文件名、磁盘路径、日志正文、归档内容和版本更新说明 MUST 保持原始内容，不得因界面语言变化而改写。

#### Scenario: 使用简体中文界面

- **WHEN** 当前实际语言为 `zh-CN`
- **THEN** 系统以简体中文展示应用自有界面、操作提示、可访问性标签与托盘菜单

#### Scenario: 使用英文界面

- **WHEN** 当前实际语言为 `en`
- **THEN** 系统以英文展示应用自有界面、操作提示、可访问性标签与托盘菜单

#### Scenario: 保留用户数据原文

- **WHEN** 用户切换界面语言且当前正在查看包含任意语言内容的日志或版本说明
- **THEN** 系统只更新应用界面文案，文件名、路径、日志正文、归档内容和版本说明保持不变

#### Scenario: 未知文案安全回退

- **WHEN** 当前语言字典缺少某个消息键或无法分类底层错误
- **THEN** 系统回退英文固定文案并保留可诊断的原始错误详情，不显示空白标签或消息键
